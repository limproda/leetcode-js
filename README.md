# LeetCode in JavaScript - Learning System

This repository documents my preparation in algorithms and data structures using JavaScript.

The goal is not to collect random solved problems, but to build a structured learning system, that helps me:

- learn core data structures and algorithms deeply
- recognize patterns instead of memorizing isolated solutions
- improve problem solving under interview conditions
- review weak areas systematically

## Philosophy

Every problem is part of a bigger learning system, that includes:

- pattern recognition
- notes
- template extraction
- mistake tracking
- periodic practice

## Problem-Solving Workflow

My typical workflow for each problem is:

1. understand the problem and constraints
2. attempt a first solution
3. identify the underlying pattern
4. refine the solution
5. document complexity, mistakes, and takeaways
